# `omniagent setup`

Initialize `~/.omniagent/omniagent.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
omniagent setup
omniagent setup --workspace ~/.omniagent/workspace
```

To run the wizard via setup:

```bash
omniagent setup --wizard
```
