# `omniagent tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
omniagent tui
omniagent tui --url ws://127.0.0.1:18789 --token <token>
omniagent tui --session main --deliver
```
