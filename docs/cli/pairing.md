# `omniagent pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
omniagent pairing list whatsapp
omniagent pairing approve whatsapp <code> --notify
```
